prometheus.prometheus.cadvisor role – cAdvisor

Note

This role is part of the prometheus.prometheus collection (version 0.26.1).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install prometheus.prometheus.

To use it in a playbook, specify: prometheus.prometheus.cadvisor.

Parameters

Parameter

Comments

cadvisor_binary_install_dir

string

Advanced

Directory to install binaries

Default: "/usr/local/bin"

cadvisor_binary_url

string

URL of the cadvisor binary file

Default: "https://github.com/{{ _cadvisor_repo }}/releases/download/v{{ cadvisor_version }}/cadvisor-v{{ cadvisor_version }}-{{ ansible_facts['system'] | lower }}-{{ _cadvisor_go_ansible_arch }}"

list / elements=string

  • "cpuset"

  • "disk"

  • "diskIO"

  • "hugetlb"

  • "memory"

  • "memory_numa"

  • "network"

  • "oom_event"

  • "percpu"

  • "perf_event"

  • "process"

  • "referenced_memory"

  • "resctrl"

  • "sched"

  • "tcp"

Default: []

boolean

list / elements=string
